# 🍔 Burger Fry Sales Dashboard & EDA Project

This project presents an **Exploratory Data Analysis (EDA)** and interactive **Excel Sales Dashboard** for **Burger Fry**, a fast-food business. The analysis focuses on sales performance, customer ordering behavior, product trends, and business insights using Excel visualization techniques.

---

## 📊 Project Highlights
- Total Sales Analysis
- Order & Revenue KPIs
- Monthly & Weekly Sales Trends
- Top-Selling Products
- Category-wise Performance
- Interactive Dashboard with Slicers
- Data Cleaning & Transformation
- Business Insights & Recommendations

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Power Query
- Dashboard Design
- Data Visualization

---

## 📈 Key Insights
- Generated over **$61K+ revenue**
- Processed **5,343 total orders**
- Average Order Value: **$11.42**
- Burgers were the highest revenue-generating category
- Weekday sales significantly outperformed weekend sales
- March recorded the highest monthly sales
